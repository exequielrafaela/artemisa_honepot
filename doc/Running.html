Running ArtemisaÂ¶

To run Artemisa write:

$ ./artemisa.py

or

$ python artemisa.py

In order to obtain the list of parameters write:

$ ./artemisa.py -h

or

$ ./artemisa.py --help

Inside the console of Artemisa, type “help” in order to get the list of available commands.

When Artemisa is executed, it registers itself to the configured SIP servers and it waits for calls. For this reason it is very important to have Artemisa properly configured before executing it.

Note

Some third-party tools used by Artemisa in order to investigate the source of the attack may require to be launched with super-user privileges. In such way, we recommend to deploy Artemisa within a virtual machine.
